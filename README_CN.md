# 汉尼可儿读心术（Hannilce Insight）

> *灵感源自《沉默的羔羊》中的审讯室——两个人隔着玻璃，互相拆解对方的防线。*
>
> **Hannilce** = Hannibal + Clarice，首尾咬合，如同一个神秘的代号。

---

## 这是什么

汉尼可儿读心术是一个 **MBTI 性格画像技能**。它不做问卷，而是通过**真实的话题对话**来读出你的性格。

在你讨论新闻话题、发表观点、表达情绪的过程中，系统逐步解析你在四个 MBTI 维度上的偏好：

| 维度 | 读什么 |
|------|--------|
| **E / I** | 外向 vs 内向 — 你从哪里获取能量 |
| **S / N** | 实感 vs 直觉 — 你如何处理信息 |
| **T / F** | 思考 vs 情感 — 你如何做决策 |
| **J / P** | 判断 vs 知觉 — 你如何对待生活秩序 |

## 如何使用

```
/MBTI <姓名>
```

示例：

```
/MBTI 张三
/MBTI 李四
```

- 每个人有独立的分析会话，互不干扰
- 重复触发同一姓名，自动从上次中断处继续（唤醒机制）

## 工作原理

```
话题生成  →  用户回复  →  质量评分  →  维度映射
(新闻/AI)    (原始文本)    (Token+语义)    (E/I S/N T/F J/P)
```

### 动态话题

- **主渠道**：每日抓取国内新闻热点（虎嗅、财新、36氪、今日头条、妈妈网）
- **降级方案**：内置 10 个话题，覆盖全部四个 MBTI 维度，采集失败时自动启用

### 质量控制

- **Token 层**：监控回复长度（100–500 tokens/轮为健康区间）
- **语义层**：独立 LLM 评分，衡量丰富度、观点鲜明度、情感真诚度
- **封存策略**：连续多轮质量下降，自动封存当前话题，不追问

### 置信度模型

```
置信度 = Token量因子 × 质量分因子 × 一致性因子
```

分析结束条件：轮次 ≥ 8 **且** 置信度 ≥ 阈值 **且** 四维均有数据覆盖。
